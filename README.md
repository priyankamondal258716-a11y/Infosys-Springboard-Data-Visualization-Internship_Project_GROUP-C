ğŸ§© Project Modules Summary
Module 1: Data Modeling and Ingestion
Cleans and structures raw hotel data into fact and dimension tables, ensuring a strong foundation for accurate analysis and reporting.

Module 2: Occupancy & Revenue Metrics
Analyzes key KPIs such as occupancy rate, revenue trends, and booking performance across time, branches, and customer segments.

Module 3: Guest Analysis
Focuses on guest segmentation, stay patterns, booking behavior, and identification of high-value customer groups.

Module 4: Forecasting & Cancellation Analysis
Uses forecasting techniques to predict future occupancy while analyzing cancellations, lead time, no-shows, and refund trends to minimize revenue loss.

Module 5: Revenue Strategy Dashboard
Builds an intelligent dashboard with dynamic pricing, occupancy sensitivity, upsell analysis, and service-based revenue insights to support revenue optimization.
